## Milestone Summary

**Key Decisions:**

- React 19 selected over Svelte 5 — Fluent UI v9 is React-only from Microsoft
- Hybrid SPA pattern — Flask renders Jinja2 shell, React mounts on #app; no CORS, no cookie reconfiguration
- ATLAS_UI=react env var toggles SPA mode — zero regression on existing deployments
- SSE via fetch + ReadableStream (not EventSource) — AbortController in useRef
- Migration order: infrastructure → functional port → visual redesign (non-negotiable)
- Tailwind v4 colon prefix syntax (tw:flex) — hyphen syntax is v3 only
- 62 --atlas- semantic tokens with three-tier surface hierarchy matching Fluent 2 webDarkTheme
- ToolPanel uses native details/summary, not Fluent Accordion
- CopyButton accepts getText: () => string for lazy clipboard evaluation (fixes DEBT-02)
- Global :focus-visible outside @layer for cascade precedence

**Issues Resolved:**

- DEBT-01: Historical messages now retain tool panels (tool events persisted to SQLite messages_json)
- DEBT-02: Copy-to-clipboard works on both streaming and historical messages
- DEBT-03: 3 test regressions fixed (description phrasing, tool count assertion, stale assertions)
- DEBT-04: get_user_photo_bytes() dead code removed
- DEBT-05: get_colleague_profile user_id schema description corrected
- auth.py url_for('index') broken by route removal — fixed to url_for('catch_all')
- ProfileCard initials fallback missing — added post-verification
- Stop button not rendering during streaming — isStreaming bridge from hook to ChatContext
- Vite dev proxy missing /chat/stream — single '/chat' proxy covers all sub-paths

**Issues Deferred:**

- login_required returns 302 instead of 401 for API routes (auth still works via fetchMe catch)
- Historical tool panels always show "Done" badge (error status not persisted to SQLite)
- Historical tool panels lose elapsed time (timestamps only in SSE events)
- AuthContext.error field set but never consumed (no flow impact)

**Technical Debt Incurred:**

- 4 non-blocking tech debt items documented in v1.2-MILESTONE-AUDIT.md
- Running badge excluded from tool panels (blocking SSE architecture prevents real-time state)
