- `test_search_users_empty_string_returns_empty`: Set `_graph_enabled = True`, call `search_users("")`, assert returns `[]`. No network call.
- `test_search_users_whitespace_only_returns_empty`: Set `_graph_enabled = True`, call `search_users("   ")`, assert returns `[]`. No network call.
- `test_search_users_sends_consistency_level_header`: Set up mocked `_cca` that returns `{"access_token": "fake-token"}` from `acquire_token_for_client`. - `test_get_user_photo_bytes_returns_none_on_404`: Mock `requests.request` to return status_code=404 with empty content. Call `get_user_photo_bytes("user-id")`. Assert returns `None`.
- `test_get_user_photo_bytes_returns_bytes_on_200`: Mock `requests.request` to return status_code=200 with `content=b"\x89PNG"`. Call `get_user_photo_bytes("user-id")`. Assert returns `b"\x89PNG"`.
- `test_get_user_photo_bytes_empty_user_id_returns_none`: Call `get_user_photo_bytes("")` with `_graph_enabled = True`. Assert returns `None`. No network call.
