<task type="auto">
  <name>Task 2: Implement async PowerShell subprocess runner with timeout and error handling</name>
  <files>
    exchange_mcp/ps_runner.py
    tests/test_ps_runner.py
  </files>
  <action>
    Create `exchange_mcp/ps_runner.py` with:

    1. Module-level constants:
       - `_PS_PREAMBLE`: Sets UTF-8 encoding and $ErrorActionPreference = 'Stop'
         ```powershell
         [Console]::OutputEncoding = [System.Text.Encoding]::UTF8
         $ErrorActionPreference = 'Stop'
         ```
       - `DEFAULT_TIMEOUT = 60`

    2. `async def run_ps(script: str, timeout: int = DEFAULT_TIMEOUT) -> str`:
       - Spawns `powershell.exe` with flags: `-NonInteractive`, `-NoProfile`, `-Command`, script
       - Uses `asyncio.create_subprocess_exec` with PIPE for both stdout and stderr
       - Wraps `proc.communicate()` in `asyncio.wait_for(timeout=timeout)`
       - On `asyncio.TimeoutError`: call `proc.kill()`, then `await proc.wait()`, then raise `TimeoutError` with descriptive message including the timeout value
       - On non-zero returncode: raise `RuntimeError` with stderr content decoded as UTF-8 (errors='replace')
       - On success: return stdout decoded as UTF-8 (errors='replace'), stripped
       - IMPORTANT: Always use `proc.communicate()` NOT `proc.wait()` to avoid pipe buffer deadlock
       - IMPORTANT: After `proc.kill()`, always `await proc.wait()` before raising

    3. `def build_script(body: str) -> str`:
       - Prepends `_PS_PREAMBLE` to the script body
       - Returns the combined script string
       - This is a pure function — no I/O

    The runner returns RAW STRING output, not parsed JSON. JSON parsing is the caller's responsibility. This keeps the runner generic.

    Create `tests/test_ps_runner.py` with unit tests:

    1. `test_run_ps_echo`: Run `run_ps('Write-Output "hello"')` and assert result contains "hello"
    2. `test_run_ps_error_exit`: Run `run_ps('exit 1')` and assert `RuntimeError` is raised
    3. `test_run_ps_timeout`: Run `run_ps('Start-Sleep -Seconds 30', timeout=2)` and assert `TimeoutError` is raised (use a short timeout)
    4. `test_build_script`: Assert `build_script("Get-Date")` starts with the UTF-8 encoding preamble
    5. `test_run_ps_utf8`: Run `run_ps('Write-Output "cafe\u0301"')` to verify UTF-8 handling (or use another non-ASCII string)

    Use `pytest` with `pytest-asyncio` for async test support. Add `pytest` and `pytest-asyncio` as dev dependencies via `uv add --dev pytest pytest-asyncio`.

    IMPORTANT: Do NOT import or reference ExchangeOnlineManagement in this file. This is the generic PowerShell runner — Exchange-specific logic lives in exchange_client.py (Plan 03).
  </action>
