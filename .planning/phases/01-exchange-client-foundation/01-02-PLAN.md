<task type="auto">
  <name>Task 1: Implement DNS TXT resolver with TTL-respecting cache</name>
  <files>exchange_mcp/dns_utils.py</files>
  <action>
    Create `exchange_mcp/dns_utils.py` with:

    1. Module-level TTL cache:
       ```python
       _cache: dict[str, tuple[list[str], float]] = {}  # name -> (records, expiry_monotonic)
       ```

    2. `async def get_txt_records(name: str) -> list[str]`:
       - Check cache first: if `name` in `_cache` and `time.monotonic() < expiry`, return cached records
       - Use `dns.asyncresolver.resolve(name, dns.rdatatype.TXT)` to query
       - Decode each rdata: `b"".join(rdata.strings).decode("utf-8")`
       - Extract TTL from `answer.rrset.ttl` (default to 300 if rrset is None)
       - Store in cache with expiry = `time.monotonic() + ttl`
       - On `dns.resolver.NXDOMAIN` or `dns.resolver.NoAnswer`: cache empty list for 300s and return `[]`
       - On other `dns.exception.DNSException`: raise `LookupError` with descriptive message

    3. `async def get_dmarc_record(domain: str) -> dict`:
       - Call `get_txt_records(f"_dmarc.{domain}")`
       - Filter for records starting with "v=DMARC1" (case-insensitive)
       - If no DMARC record found: return `{"found": False, "domain": domain}`
       - If found: return `{"found": True, "domain": domain, **parse_dmarc(record)}`

    4. `async def get_spf_record(domain: str) -> dict`:
       - Call `get_txt_records(domain)`
       - Filter for records starting with "v=spf1" (case-insensitive)
       - If no SPF record found: return `{"found": False, "domain": domain}`
       - If found: return `{"found": True, "domain": domain, **parse_spf(record)}`

    5. `def parse_dmarc(txt_record: str) -> dict`:
       - Split on `;\s*` to extract tag-value pairs
       - Return dict with keys: version, policy (p), subdomain_policy (sp, defaults to p), pct (int, defaults to 100), rua, ruf, adkim (defaults to "r"), aspf (defaults to "r"), raw
       - Reference: RFC 7489 tag-value format

    6. `def parse_spf(txt_record: str) -> dict`:
       - Split on whitespace, skip "v=spf1" token
       - Collect mechanisms (include:, a:, mx:, ip4:, ip6:, etc.)
       - Extract all-qualifier (+all, -all, ~all, ?all)
       - Return dict with keys: version ("spf1"), mechanisms (list), all (qualifier string or None), raw
       - Reference: RFC 7208

    7. `def clear_cache() -> None`:
       - Clear the `_cache` dict. Exposed for testing.

    IMPORTANT: Use system default DNS resolver. Do NOT configure custom nameservers.
    IMPORTANT: DKIM is deferred to Phase 5. Only DMARC and SPF in this module for now.
  </action>
  <verify>
    - File exists and imports cleanly: `uv run python -c "from exchange_mcp.dns_utils import get_dmarc_record, get_spf_record, parse_dmarc, parse_spf"`
    - parse_dmarc and parse_spf are pure functions that can be tested without network
  </verify>
