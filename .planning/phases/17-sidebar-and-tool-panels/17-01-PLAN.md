Example for each call site:

```python
start_ts = time.time()
try:
    result_text = call_mcp_tool(tool_name, arguments)
    tool_events.append({
        "name": tool_name,
        "status": "success",
        "params": arguments,
        "result": result_text,
        "start_time": start_ts,
        "end_time": time.time(),
    })
except Exception as exc:
    result_text = f"Tool error: {exc}"
    tool_events.append({
        "name": tool_name,
        "status": "error",
        "params": arguments,
        "result": result_text,
        "start_time": start_ts,
        "end_time": time.time(),
    })
    logger.warning("MCP tool %s failed: %s", tool_name, exc)
```

**IMPORTANT:** Both except blocks already contain `logger.warning("MCP tool %s failed: %s", tool_name, exc)` lines. PRESERVE those logger.warning() calls — do not remove them. Only add the `start_ts` variable and the two new dict keys.
