MODIFY chat_app/static/app.js:

1. ADD a copyText utility function inside the IIFE, before createAssistantMessage:

```javascript
function copyText(text, btn) {
    if (!navigator.clipboard) return;
    navigator.clipboard.writeText(text).then(function() {
        var original = btn.textContent;
        btn.textContent = 'Copied!';
        btn.classList.add('copy-success');
        btn.disabled = true;
        setTimeout(function() {
            btn.textContent = original;
            btn.classList.remove('copy-success');
            btn.disabled = false;
        }, 1500);
    }).catch(function(err) {
        console.error('[Atlas] Copy failed:', err);
    });
}
```

2. MODIFY createAssistantMessage() — add a copy button for the AI response text:

After the cursor is appended to els.content and before the wrapper is appended to messagesEl, create a copy button:

```javascript
var copyBtn = document.createElement('button');
copyBtn.className = 'copy-btn';
copyBtn.type = 'button';
copyBtn.title = 'Copy response';
copyBtn.textContent = 'Copy';
copyBtn.addEventListener('click', function(e) {
    e.stopPropagation();
    // textNode is the Text node where streamed text accumulates.
    // Its .textContent naturally contains only the AI answer text —
    // tool panels are separate DOM elements, not children of textNode,
    // so no filtering is needed.
    copyText(textNode.textContent, copyBtn);
});
els.content.appendChild(copyBtn);
```

The copy button is appended AFTER the cursor span. It will be hidden by default and revealed on hover via CSS.
