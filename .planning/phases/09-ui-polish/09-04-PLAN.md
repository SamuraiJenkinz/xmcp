MODIFY chat_app/templates/base.html:

1. ADD flash-prevention inline script in the `<head>` BEFORE the `<link rel="stylesheet">` line:

```html
<script>
(function() {
    'use strict';
    var stored = localStorage.getItem('atlas-theme');
    if (stored === 'dark' || stored === 'light') {
        document.documentElement.setAttribute('data-theme', stored);
    } else if (window.matchMedia && window.matchMedia('(prefers-color-scheme: dark)').matches) {
        document.documentElement.setAttribute('data-theme', 'dark');
    }
})();
</script>
```

This MUST be placed BEFORE `<link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">` so the data-theme attribute is set before any CSS is parsed and applied. This prevents the white flash for dark mode users.
