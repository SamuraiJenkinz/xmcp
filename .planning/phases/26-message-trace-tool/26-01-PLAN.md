        if subject is None:

        if len(subject) <= max_len:

        return subject[:max_len] + "..."

    **C. Handler function** — add before the TOOL_DISPATCH dict, following the exact pattern from _check_mail_flow_handler. Use the full handler skeleton from 26-RESEARCH.md (the Code Examples section) as the implementation. Key requirements:

    1. Validate `client is not None`
    2. Extract and strip sender_address and recipient_address
    3. Require at least one of sender OR recipient (raise RuntimeError if both empty)
    4. Reject bare names without "@" (raise RuntimeError with guidance)
    5. Validate full email format using `_validate_upn` for addresses with exactly one "@" that don't start with "@"
    6. Date handling: default to last 24 hours; parse provided dates with `datetime.fromisoformat`; validate 10-day max range
    7. Result size: default 100, cap at 1000; request `min(result_size + 1, 5000)` from PowerShell for truncation detection
    8. Build Get-MessageTraceV2 cmdlet string using `_escape_ps_single_quote` for all string params
    9. Include `-Subject` and `-SubjectFilterType 'Contains'` only when subject_filter provided
    10. Pipe to `Select-Object SenderAddress, RecipientAddress, Received, Status, Subject, MessageTraceId, Size, FromIP, ToIP, ConnectorId`
    11. Append `| ConvertTo-Json -Depth 10` at the end of the cmdlet
    12. Call `await client.run_cmdlet_with_retry(cmdlet)`
    13. Handle RuntimeError: if error message contains "not found" patterns, return empty result dict
    14. Normalize: if raw is dict (single result), wrap in list; if raw is falsy, empty list
    15. Detect truncation: `len(results_raw) > result_size`; slice to result_size
    16. Map each result to output dict with: sender, recipient, received, status, subject_snippet (via _truncate_subject), message_trace_id, size_kb (Size / 1024, rounded to 1 decimal), from_ip, to_ip, connector
    17. Return dict with: results (list), count (int), truncated (bool), query_summary (sender, recipient, start_date ISO, end_date ISO)
